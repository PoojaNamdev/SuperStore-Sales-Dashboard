🚀 SuperStore Sales Dashboard | Power BI Project
📊 Project Overview

This project presents a complete end-to-end data analysis solution using Power BI.
The dashboard transforms raw sales data into meaningful insights through data cleaning, modeling, and visualization.

It also includes a 15-day sales forecasting model to predict future trends and support business decision-making.

📌 Key Metrics
💰 Total Sales: 1.57M
📈 Total Profit: 175K
📦 Total Quantity Sold: 22K
🚚 Average Shipping Time: 4 Days

🔎 Key Insights
📍 California generated the highest sales (~0.34M)
🏆 Office Supplies is the top-performing category
📅 October recorded one of the highest sales peaks
🚚 Standard Class is the most frequently used shipping mode

📈 Dashboard Features
Interactive and dynamic visuals
Region-wise and state-wise sales analysis
Category & sub-category performance tracking
Monthly sales and profit trends
Sales forecasting (15 Days)
Filters and slicers for deep data exploration
